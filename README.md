# GenLayer Educational Content

Learn to build **Intelligent Contracts and apps** on GenLayer with tutorials, diagrams, livestream notes, and code examples.

## Repository Structure
- `tutorials/` – Step-by-step guides on GenLayer features
- `code-examples/` – Sample contracts and validator scripts
- `diagrams/` – Architecture and process visuals
- `livestreams/` – Notes or recordings from GenLayer sessions

## Getting Started
1. Open this repository in **GitHub Codespaces** (green **Code → Open with Codespaces**)
2. Explore tutorials, code examples, and diagrams
3. Run Python or Solidity code examples directly in the Codespace

## Contribution
Add your own tutorials, visuals, or code examples to help others learn GenLayer.
