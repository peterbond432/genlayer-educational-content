# GenLayer Educational Content 🚀

A beginner-friendly repository to learn how to build **Intelligent Contracts and applications on GenLayer**.

This project provides step-by-step tutorials, practical code examples, and visual learning resources to help developers quickly understand and start building on GenLayer.

---

## 📚 Repository Structure

- `tutorials/` – Step-by-step guides on GenLayer features and concepts  
- `code-examples/` – Sample scripts and validator simulations  
- `diagrams/` – Architecture and workflow visuals  
- `livestreams/` – Notes or recordings from GenLayer sessions  

---

## 🚀 Getting Started

1. Open the repository in **GitHub Codespaces**
2. Navigate to the `code-examples/` folder
3. Run the example:

```bash
python code-examples/hello_world_validator.py
